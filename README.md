## 📖 Overview

**CampusBuzz** is a full-stack web application designed to make campus communication smarter and more connected.  
Students, faculty, and admins can share updates, events, announcements, and placements — all in one centralized platform.  

Say goodbye to messy WhatsApp groups and notice boards 👋  
CampusBuzz brings everything under one clean, responsive, and modern interface.

---

## ✨ Features

✅ **User Authentication** – Secure signup/login using JWT  
✅ **Role-Based Access** – Separate dashboards for Students, Faculty & Admin  
✅ **Announcements Feed** – Campus events, placements, and news  
✅ **Interactive UI** – Beautiful design using React + TailwindCSS  
✅ **RESTful APIs** – Powered by Node.js + Express  
✅ **MongoDB Integration** – Fast and scalable data handling  
✅ **Responsive Design** – Works on all devices  
✅ **Deployment Ready** – Compatible with Render, Hostinger & Vercel

## 🧠 Installation & Setup

## 1️⃣ Clone the Repository
git clone https://github.com/mansi-24082006/Campus_Buzz.git
cd Campus_Buzz

##2️⃣ Install Dependencies

Frontend:
cd frontend
npm install

Backend:
cd ../backend
npm install

##3️⃣ Configure Environment Variables
Create a .env file inside the backend folder:
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=8800

##4️⃣ Run the Project

Frontend:
cd frontend
npm run dev


Backend:
cd backend
npm start

##🖥️ Frontend → http://localhost:5173

##⚙️ Backend → http://localhost:8800

🧰 Build for Production
cd frontend
npm run build

##🌍 Deployment

Frontend: Deploy the dist/ folder to GitHub Pages or Vercel
Backend: Deploy your server to Render / Hostinger / Railway
Database: Use MongoDB Atlas (cloud-hosted MongoDB)
